## 📱 주요 기능

| 기능 | 설명 |
|------|------|
| 🔍 **돋보기** | 실시간 카메라 확대 · 확대 촬영 · 텍스트 인식 |
| 💊 **복약 관리** | 약 복용 알림 · 복약 기록 관리 |
| 🏥 **병원 찾기** | 주변 병원·약국 검색 · 긴급 전화번호 |
| 🆘 **긴급 SOS** | 원터치 긴급 발신 · 가족 연락처 등록 |
| 🧠 **두뇌 게임** | 치매 예방 · 기억력·집중력 훈련 |
| 🩺 **건강 기록** | 혈압·혈당·체중·체온 기록 |
| 📰 **큰글씨 뉴스** | 글자 크기 조절 · 오늘의 뉴스 |
| 🚌 **대중교통** | 길찾기 · 65세 이상 무료 이용 안내 |
| 📺 **화면공유** | 자녀가 부모님 화면 실시간 보기 |
| 👨‍👩‍👧 **가족 연동** | 자녀가 부모님 복약·건강 현황 확인 |

---

## 🛠 기술 스택

- **Frontend**: Next.js 14.2.5, React 18
- **Database**: Supabase
- **Auth**: Supabase Auth
- **화면공유**: WebRTC (P2P)
- **배포**: Vercel
